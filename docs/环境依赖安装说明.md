# 环境依赖安装说明

## VC++ 运行库

CapsWriter-Offline 依赖 VC++ 运行库。如果启动时提示缺少 DLL，说明系统未安装。

**下载安装**：<https://aka.ms/vs/17/release/vc_redist.x64.exe>

直接下载安装即可，一般安装一次后系统里的所有程序都能用。


## FFmpeg

文件转录功能需要调用 `ffmpeg`（以及可选的 `ffprobe`）来提取音视频文件的音频流。

### 推荐做法：内置到程序目录（零配置）

直接将 `ffmpeg.exe` 和 `ffprobe.exe` 拷贝到程序根目录（与 `start_server.exe` 同级）。程序会优先从该目录加载，无需任何环境变量配置。

这对分发给终端用户最友好——用户不需要自己装 FFmpeg。

### 次选做法：安装到系统 PATH

如果不想内置，也可以让用户自行安装：

1. 访问 <https://ffmpeg.org/download.html>，点击 Windows 下的图标
2. 推荐从 **gyan.dev** 下载：<https://www.gyan.dev/ffmpeg/builds/ffmpeg-git-full.7z>
3. 解压后，将 `ffmpeg/bin` 文件夹路径添加到系统 PATH 环境变量中

### 解析优先级

程序内部通过 `core/tools/ffmpeg_resolver.py` 解析 ffmpeg/ffprobe 路径，优先级：

1. **程序根目录下的 `ffmpeg.exe` / `ffprobe.exe`**（PyInstaller 打包后即 exe 同目录；开发模式即脚本同目录）
2. 系统 PATH（`shutil.which` 兜底）

### ffprobe 的作用

`ffprobe` 非必需，但缺少时：
- 文件转录进度条只能显示已发送时长，无法显示总进度
- 不影响转录结果

建议随 `ffmpeg.exe` 一起内置 `ffprobe.exe`。
